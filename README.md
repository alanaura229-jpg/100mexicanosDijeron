Quiz Time – Trivia 🎯
Juego de trivia para escritorio hecho en C# como proyecto de la materia de Interfaces Gráficas. 
Las preguntas vienen de una base de datos en MySQL y el jugador tiene que elegir entre 4 opciones por pregunta.

Instrucciones 
Eliges una categoría (o modo aleatorio), respondes 10 preguntas y al final te dice cuántas acertaste. Hay preguntas normales de texto y también algunas donde las opciones son imágenes.
Las categorías disponibles son:
- Música
- Deportes
- Geografía y Viajes
- Cine y Series
- Videojuegos y Tecnología

Implementado en: 
- Visual Studio: para programar todo en C#
- MySQL: la base de datos con las preguntas y categorías
- Canva: para diseñar las pantallas e imágenes
- GitHub: control de versiones y trabajo en equipo
